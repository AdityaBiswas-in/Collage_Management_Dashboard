<H1>🎓 College Management Dashboard</H1>  

A fully interactive and editable College Management Dashboard built using HTML, CSS, and JavaScript. This project simulates a real-world college system where administrators can manage departments, semesters, students, subjects, and attendance efficiently.

<H1>📌 Features</H1>

<H2>🏫 Department Management</H2>  

Add and manage multiple departments (BCA, BBA, B.Sc, etc.)

Each department dynamically contains its own semesters

--- 

<H2>📚 Semester System</H2> 

Automatic generation of semesters (6 or 8 based on course)

Structured navigation inside each department

---

<H2>👨‍🎓 Student Management</H2> 

Add students with details

Assign students to specific semesters

---

<H2>📖 Subject Management</H2> 

Add and manage subjects per semester

Organized subject allocation system

---

<H2>📝 Attendance System</H2> 

Mark attendance for students

Track and manage attendance records

---

<H2>⚙️ Fully Editable Dashboard</H2> 

Add, update, and manage all data dynamically

User-friendly interface for real-time changes

---

<H2>🎨 UI/UX</H2> 

Clean and responsive design

Dashboard-style layout for better usability

---
